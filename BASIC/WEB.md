## CSS
用于规定HTML的样式
样式引用规则：
- 优先级：内联>内部>外部
- 如果有多重样式的引用，则符合继承规则
### 内联样式
标签属性设置
### 内部样式
使用`<style></style>`标签设置
### 外部样式
使用`<link></link>`标记css文件
标签的id等于css中引用的样式名
